# Hanappe Framework

## Introduction
This is the framework for Moai SDK.

http://getmoai.com/

## Feature
MoaiSDK is powerful.
However, there are many little cumbersome.  

You will be easily able to use the Hanappe.

* To simplify the coding
* Based object-oriented
* High performance
* Can mix standard code MoaiSDK
